# lalrpop.vim

This is a Vim plugin for [LALRPOP](https://github.com/nikomatsakis/lalrpop) with minimal syntax-highlighting functionality.

## Dependency

It depends on [rust.vim](https://github.com/rust-lang/rust.vim).
